dsrpg by wilminator

PHP/HTML/CSS/Javascript RPG Engine

Dragon Star RPG Engine

Dependencies

MySQL 4 or better PHP 5 HTML 4 compliant web browser Flash 9 or better for audio Authenticate project http://github.com/wilminator/authenticate

Install

Install the Authenticate project onto your web server. Copy the source code for this project to a folder in your htdocs folder. In your MySQL server, create a database and a user account for this program. Ensure the account can create, delete, insert, select, lock tables and index the database. Edit the include/mysql.cfg file to contain the database to use, the mysql host name, the mysql user account, and the mysql password to use. Open your web browser and have fun!
